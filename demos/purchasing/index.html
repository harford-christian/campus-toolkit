Attention Classroom Teachers:

Please keep classroom supply and enhancement requests for the 2026-2027 school year reasonable. Basic classroom furniture (filing cabinet, bookshelf) and consumable supplies (paperclips, file folders, etc.) are separate. If a request needs to run high (for example a science teacher ordering dissections or an art teacher ordering supplies), just include a short note explaining the need when you submit to purchasing.
